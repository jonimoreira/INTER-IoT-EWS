rdfs:comment
An event is a trigger that can be used to synchronize two or more activities. Events and their properties are further specialized in the LogiServ ontology with URL https://raw.githubusercontent.com/danielelm/tno/master/ontology/LogiServ.tt With this definition of event in LogiCO, we specialize the concept of Event defined in a general way (on purpose) in the DOLCE UltraLite (DUL) ontology. According to the DUL definition, an event consists of any physical, social, or mental process, event, or state.
