Class LogiCO:Parameter

rdf:type
owl:Class
rdfs:comment
A Concept that classifies an Abstract, such as, for example, a Physical Attribute, a Space Region or a Temporal Region. The difference between an Abstract and a Parameter is that abstracts represent sets of observable values, e.g. 'the height of a given building', while parameters represent constraints or selections on observable values, e.g. 'VeryHigh'. Equivalent to the concept of Parameter in DOLCE UltraLite (DUL) ontology.
